# Interactive course app

Generated by the `repo-learning-builder` skill. A React + TypeScript + Vite
single-page app that renders a compact course (`src/data/course.json`) into a
premium, self-paced learning experience. The app is a permanent template; only
the course data changes per project.

## Commands

```bash
npm install      # install dependencies
npm run dev      # dev server (http://localhost:5173)
npm run test     # unit + jsdom interaction tests
npm run build    # typecheck + static build in dist/
npm run preview  # serve the production build
```

The production build uses relative asset paths, so `dist/` works on any static
host or from a file path.

## Structure

- `src/data/course.json` compact v2 course content (data, not code).
- `src/lib/` framework-free logic: `schema.ts` (types + validation), `expand.ts`
  (compact -> runtime), `archetypes.ts`, `theme.ts`, `progress.ts`, `mastery.ts`,
  `navigation.ts`, `stats.ts`, `regen.ts`, `sources.ts`, `search.ts`, `storage.ts`,
  `markdown.ts`. Unit-tested in `src/lib/__tests__`.
- `src/components/` the UI: cover home + learning path, lesson view with facet
  switcher / callouts / walkthrough, repository explorer, progress dashboard with
  a concept radar, quiz engine, glossary, search, certificate.
- `src/context/AppContext.tsx` wires the course + progress store into React and
  handles hash routing, theme, and level.

## Themes

Four curated themes (Explorer, Laboratory, Storybook, Blueprint) in `index.css`,
selected from the course category or by the learner. Light and dark are designed
independently. Set an accent, motif, or icon via `theme` in `course.json`.

## Progress and privacy

Progress is stored in the browser (`localStorage`) keyed by course id. No account
or server for the default mode. Export, import, reset, and a shareable summary
live on the Progress dashboard. Keyboard: arrow keys move between lessons.

## Regenerating

Replace `course.json`. Keep lesson and concept ids stable to preserve progress
(the store migrates by id). Reduced motion, print, and 360px-to-desktop layouts
are built in.
